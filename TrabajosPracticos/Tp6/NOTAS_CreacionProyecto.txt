Para crear el proyecto de React con Vite.js ejecutamos el comando:
npm create vite@latest

Luego podremos definir nombre y tecnologias a utilizar en la CLI de Vite.js


Para instalar las dependencias del proyecto: npm install
Para ejecutar el proyecto en modo desarrollo: npm run dev



GUIA Nueva Branch:

Creamos la branch local usando el comando: git checkout -b <nombre-branch>

Realizamos los cambios y commits.

Para pushear a una branch remota se ejecuta el comando:
git push -u origin <nombre-branch>


Si la branch YA esta creada, listamos las branches remotas: git branch -r
Para cambiar a la branch remota se ejecuta: git checkout <nombre-branch-remota>
